Indicaciones de uso

  1. Completá el nombre del evento, el presupuesto y la fecha si querés dejar esos datos guardados.
  2. Agregá al menos 3 participantes. Los nombres no pueden repetirse.
  3. Si dos personas no pueden tocarse, agregalas en la sección de restricciones.
  4. Presioná Realizar sorteo. La app mezcla los nombres y evita autoasignaciones.
  5. Copiá el link privado de cada participante y mandáselo solo a esa persona.
  6. Cada participante abre su link y ve únicamente a quién le toca regalarle.
Compartir

Resultados privados

Después del sorteo aparece una lista de links privados. Cada link contiene solo el resultado de una persona, por eso se puede abrir desde otro navegador, celular o computadora sin que se reinicie.

Como la app no usa servidor, quien organiza técnicamente podría abrir cualquier link antes de enviarlo. Para un sistema donde ni la persona organizadora pueda saber los resultados haría falta un backend.

FAQ

Preguntas frecuentes

¿Por qué al abrir la app en otro navegador aparece vacía?

La lista principal se guarda localmente en el navegador donde se creó. Para compartir resultados con otras personas usá los links privados que aparecen después de realizar el sorteo.

¿Los participantes pueden ver todos los resultados?

No desde su link privado. Cada link muestra únicamente el resultado de esa persona.

¿Qué pasa si alguien pierde su link?

La persona organizadora puede volver a la app desde el mismo navegador donde hizo el sorteo y copiar nuevamente el link privado.

¿Puedo cambiar participantes después de sortear?

Sí, pero al editar, agregar o eliminar participantes se limpia el sorteo anterior. Después tenés que realizar un nuevo sorteo.

¿Qué son las restricciones?

Son pares de personas que no pueden tocarse entre sí. Sirven para parejas, familiares o cualquier combinación que quieras evitar.

¿Puedo descargar los resultados?

Sí. Después de sortear podés descargar un archivo JSON con la información del evento, restricciones y asignaciones.

¿Funciona en GitHub Pages?

Sí. La app es estática y funciona con HTML, CSS y JavaScript, sin backend ni instalación.
